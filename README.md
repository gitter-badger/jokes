jokes
=====

Some especial programmers humour («программисты шутят»).

  * Wrong keyboard layout
    - ``this`` - ``ершы``
    - ``self`` - ``ыуда``
    - ``reboot`` - ``куищще``
    - ``WHERE`` - ``ЦРУКУ``
    - ``bash`` - ``ифыр``
    - ``git`` - ``пше``
    - ``github`` - ``пшерги``
    - ``<pre>`` - ``БзкуЮ``
    - ``DNS`` - ``ВТЫ``
    - ``server`` - ``ыукмук``
    - ``vim`` - ``мшь``
    - ``devel`` - ``вымуд``
  * Wrong cyrillic charset
    - ``Вопрос`` - ``бНОПНЯ``
    - ``Вопрос читал?`` - ``бНОПНЯ ВХРЮК?``
    - ``Сергей`` - ``яЕПЦЕИ``
  * Mistyped
    - ``<input type="button" onlick="..." />``
  * Machine translation
    - ``master/slave`` - ``барин/холоп``
