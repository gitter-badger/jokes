jokes
=====

Some especial programmers humour («программисты шутят»).

  * Wrong keyboard layout
    - ``this`` - ``ершы``
    - ``self`` - ``ыуда``
    - ``reboot`` - ``куищще``
    - ``WHERE`` - ``ЦРУКУ``
    - ``bash`` - ``ифыр``
  * Wrong cyrillic charset
    - ``Вопрос`` - ``бНОПНЯ``
    - ``Вопрос читал?`` - ``бНОПНЯ ВХРЮК?``
    - ``Сергей`` - ``яЕПЦЕИ``
